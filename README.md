# Sentiment Analysis of Reviews

## Project Description
This project classifies movie reviews as Positive or Negative using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## Accuracy
83.58%

## Dataset
IMDB Movie Reviews Dataset

## Features
- Text preprocessing
- Sentiment classification
- Accuracy evaluation
